Device Options:
Uface - tick to handle comms with new platform device's TC101, MB800, uface402 or any device with new menu style.
Oldtime - tick to resolve time by one hour, use for prox terminals or uface. If time is out by one hour change this option.
Prox - tick to handle comms with prox terminals to prevent repolling
NoPhoto - tick to remove Photo command being built.
NoFace - tick to remove Face Template commands being built.
NoFinger - Tick to remove finger commands being built.
NoTime - tick to prevent time header being sent, use when setting server time is not suitable.
Timezone - Enter plus or minus figure to change the time according to device local Timezone
